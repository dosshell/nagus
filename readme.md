# Nagus

usage:
```
nagus add package_name-2.7.3
nagus add gandnagus.json

nagus add user:password@\\smbserver\share\folder\
nagus add /folder/something/
nagus add c:\folder\

nagus rm python-2.7.3
nagus rm c:\folder\
nagus rm *

nagus keep ../
nagus keep python-2.7.3 opencv-1.2.3
nagus keep nagus.json

nagus list servers
nagus list packages
nagus list nagus.json

nagus stash c:\nagus_stash\
```
